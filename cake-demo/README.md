# Cake Configurator — конструктор торта для Iryna (#27)

Один HTML-файл. **Без бекенду. Без хостингу. Без коду.** Замовлення приходить майстру в Telegram через `t.me/?text=`.

## Як це працює

1. Клієнт відкриває сторінку
2. Обирає вагу / начинку / декор
3. Бачить ціну в реальному часі
4. Заповнює ім'я + контакт + дату
5. Тисне **"Замовити через Telegram"**
6. Відкривається Telegram з готовим форматованим повідомленням → клієнт відправляє його кондитеру

## Деплой за 30 секунд (Netlify Drop)

1. Відкрий https://app.netlify.com/drop
2. Перетягни папку `cake-demo` в зону на сайті
3. Через 5 секунд отримаєш URL типу `https://elegant-cake-12345.netlify.app`
4. Все, працює. Без реєстрації, безкоштовно назавжди.

Для постійного URL — створи безкоштовний акаунт Netlify, можеш привʼязати свій домен (типу `irynacake.com.ua` за ~200 грн/рік).

## Налаштування під клієнта

Відкрий `index.html`, у верху знайди блок `CONFIG`:

```js
const CONFIG = {
  telegram: "tort_vladimirovna",      // <— її @username (без @)
  businessName: "Tort Vladimirovna",  // <— назва кондитерської
  city: "Тернопіль",                  // <— місто
  heroImage: "https://...",           // <— URL фото з її IG
  basePricePerKg: 650,                // <— базова ціна за кг
  fillings: { ... },                  // <— список начинок та надбавок
  decorOptions: { ... },              // <— список декору
  expressSurchargePercent: 30,
  expressThresholdDays: 3
};
```

### Як підставити її фото

1. Відкрий https://www.instagram.com/tort_vladimirovna/
2. Правий клік на фото → "Зберегти зображення як" → завантаж файл
3. Загрузи фото в безкоштовний хостинг типу:
   - [imgur.com](https://imgur.com) — drag&drop, отримуєш URL
   - Або просто закинь файл у папку `cake-demo` і пиши `heroImage: "moe-foto.jpg"`

### Як уточнити ціни під неї

Подивись її IG — там часто є коментарі типу "торт від 600 грн/кг" або питання в директ "скільки коштує". Подзвони і питай:
- "Яка ваша базова ціна за кг?" → `basePricePerKg`
- "Які начинки? І чи є надбавка за більш складні?" → `fillings`
- "Що додатково за квіти / фігурки / напис?" → `decorOptions`

Якщо вона не каже — постав ринкові: 600-700/кг базова, начинки +0-100, декор 150-350.

## Кастомізація під інші ніші

Цей самий шаблон легко переробляється під:
- **Барбершопи / манікюр** — заміни "вага / начинка / декор" на "послуга / майстер / час"
- **Фотографи** — "тип зйомки / локація / тривалість / дата"
- **Йога** — "абонемент 8/12 занять / напрямок / місяць"
- **Репетитори** — "предмет / рівень / кількість занять / дата старту"
- **СТО** — "марка авто / тип проблеми / коли можна привезти"

Логіка та сама: форма → live ціна → відкриття Telegram з готовим повідомленням.
