# mini-sites 🚀

Мікро-сайти для малого бізнесу України. Кожен — один HTML-файл, без бекенду, безкоштовний хостинг.

## Клієнти

| # | Папка | Ніша | Місто | Статус |
|---|-------|------|-------|--------|
| 1 | [cake-demo](./cake-demo/) | 🎂 Кондитер | Тернопіль | ✅ Готово |

## Як запустити локально

```bash
python -m http.server 8080 --directory cake-demo
# відкрий http://localhost:8080
```

## Деплой

Netlify Drop → перетягнути папку клієнта на https://app.netlify.com/drop
